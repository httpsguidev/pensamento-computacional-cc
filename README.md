# Pensamento Computacional

Este repositório tem como objetivo documentar a aplicação prática dos pilares do pensamento computacional ao longo da disciplina, por meio de atividades, exercícios e projetos desenvolvidos durante as aulas.

A proposta é utilizar uma abordagem estruturada para resolução de problemas, transformando desafios complexos em soluções organizadas, eficientes e reutilizáveis.

## Objetivos

- Desenvolver a capacidade de resolver problemas de forma lógica e estruturada
- Aplicar os pilares do pensamento computacional em diferentes contextos
- Criar soluções que possam ser replicadas, escaladas ou automatizadas
- Registrar a evolução do aprendizado ao longo da disciplina

## Pilares do Pensamento Computacional

### 1. Decomposição

Dividir problemas complexos em partes menores e mais gerenciáveis.

Aplicação:
Cada atividade será organizada em etapas, facilitando o entendimento e a implementação gradual da solução.

### 2. Reconhecimento de Padrões

Identificar similaridades entre problemas para reutilizar soluções já conhecidas.

Aplicação:
Comparação entre exercícios e identificação de estratégias reutilizáveis.

### 3. Abstração

Focar apenas nos elementos essenciais do problema, ignorando detalhes irrelevantes.

Aplicação:
Simplificação de problemas para destacar sua lógica principal antes da implementação.

### 4. Algoritmos

Desenvolver passos ordenados para resolver um problema.

Aplicação:
Criação de fluxos lógicos, pseudocódigo e implementações práticas.

## Metodologia

Cada atividade seguirá um padrão:

1. Descrição do problema
2. Decomposição
3. Abstração
4. Padrões identificados
5. Algoritmo ou solução
6. Implementação (quando aplicável)
7. Reflexão

## Evolução

Este repositório será atualizado continuamente conforme o avanço da disciplina, servindo como registro de aprendizado e base para revisão de conteúdo.

## Conexão com a Computação

O pensamento computacional é uma habilidade essencial para o desenvolvimento de software, análise de sistemas e modelagem de soluções. Ele permite estruturar problemas de forma que possam ser resolvidos de maneira eficiente, tanto por humanos quanto por máquinas.
